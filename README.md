<h2>Install/Kurulum</h2>

<table>
  <tr>
    <th>Steps/Adımlar</th>
  </tr>
  <tr>
    <td>
      1.) "MyScreenSaver.src" dosyasına sağ tıklayın. <br>
      1.) Right-click on the "MyScreenSaver.src" file.
    </td>
  </tr>
  <tr>
    <td>
      2.) Yükle'ye tıklayın. <br>
      2.) Click to "Install".
    </td>
  </tr>
  <tr>
    <td>
      3.) Süre'yi 1 dk olarak ayarlayın.<br>
      3.) Set the time to 1 min.
    </td>
  </tr>
</table>
